Clientes

Listagem

A listagem é feita na página de Clientes. Para acessá-la é necessário login.

Cadastro

Para cadastrar um cliente, após feito login, vá até a página Clientes, clique no botão "Cadastrar Cliente" (ou acessar a url /cadastro-cliente) e em seguida preencha os campos do formulário.

Edição e remoção

Para editar ou remover um cliente, vá até a listagem de clientes, clique no cliente desejado e escolha se deseja editá-lo ou removê-lo. Caso escolha editar, será aberto o formulário de edição das informações. Caso escolha remover, o cliente simplesmente será removido.
